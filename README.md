# Task Manager App

A full-stack Task Manager application built using modern web technologies.

---

## 🚀 Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: MongoDB Atlas
- Deployment: Render

---

## ✨ Features

- User Signup & Login (JWT Authentication)
- Create Tasks
- View Tasks
- Add Multiple Users
- Secure Backend API
- Fully Deployed Backend

---

## ⚙️ Installation

### 1. Clone Repository
git clone https://github.com/ArshadGhodake/task-manager.git

---

### 2. Install Dependencies

Backend:
cd backend
npm install

Frontend:
cd frontend
npm install

---

## ▶️ Run Project

Backend:
cd backend
npm start

Frontend:
cd frontend
npm start

---

## 🔐 Environment Variables (Backend)

Create a `.env` file inside backend folder:

MONGO_URI=your_mongodb_connection_string  
JWT_SECRET=your_secret_key  

---

## 📌 Notes

- MongoDB Atlas is used (cloud database)
- node_modules is ignored via .gitignore
- Backend is deployed on Render
- CORS enabled for frontend-backend connection
